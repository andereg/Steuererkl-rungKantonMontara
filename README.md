# Steuererklärung Classic Site

Statische Demo für eine klassische Steuererklärungs-Webseite mit:

- `index.html`
- `style.css`
- `script.js`

## Enthalten
- persönliche Angaben
- Einkommen
- Abzüge
- Bemerkungen
- automatische Zusammenfassung
- Standardabzüge
- lokales Speichern im Browser (`localStorage`)

## Start
Einfach `index.html` im Browser öffnen.

## Hinweis
Diese Demo ist rein frontend-basiert und sendet nichts an einen Server.
